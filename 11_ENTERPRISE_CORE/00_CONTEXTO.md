# 00. Contexto del Proyecto

## Información General
Este documento sirve como registro del contexto y las entradas para la planeación del proyecto, cumpliendo con los requisitos de documentación de ISO 9001.

## Entradas de Planeación (Input)
*   **Origen**: Sincronización v1.8 (Definición Estratégica y Análisis Profundo).
*   **Versión del Contexto**: v1.8
*   **Fecha de Corte**: 2026-01-01

## Filosofía Operativa: "Ritmo Humano, Potencia Máquina"
*   **Control de Ritmo**: El software JAMÁS cierra una fase automáticamente. Es el **humano** quien decide explícitamente cuándo está listo para avanzar.
*   **Análisis Profundo (Prompt 2)**: Se introduce una fase dedicada exclusivamente al análisis estratégico (FODA+CAME) utilizando la capacidad de contexto largo de AntiGravity.
*   **Granularidad de Análisis**: Para temas complejos, el análisis se exige por **bloques de 100 líneas** (o unidades lógicas equivalentes) para evitar generalidades.

## Restricciones y Reglas de Flujo
*   **Separación Análisis vs. Ejecución**: No se ejecuta nada hasta que el análisis estratégico (Prompt 2) esté completo y validado.
*   **Validación Cruzada**: Se fomenta el uso opcional de otros modelos de IA para contrastar el FODA generado por AntiGravity.
*   **Plataforma**: Windows Desktop (.exe).
