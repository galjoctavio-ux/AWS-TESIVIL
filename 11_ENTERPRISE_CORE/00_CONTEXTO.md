# 00. Contexto del Proyecto

## Información General
Este documento sirve como registro del contexto y las entradas para la planeación del proyecto, cumpliendo con los requisitos de documentación de ISO 9001.

## Entradas de Planeación (Input)
*   **Origen**: Sincronización v1.5 (Definición Pedagógica y de Alcance).
*   **Versión del Contexto**: v1.5
*   **Fecha de Corte**: 2025-12-24

## Filosofía Operativa Refinada
*   **Dependencia Consciente**: Se asume explícitamente que **AntiGravity** es el motor de ejecución. El software prepara el trabajo para este actor específico.
*   **Enfoque Didáctico**: El software no solo "hace", sino que **enseña al usuario a dirigir**. El objetivo es que el usuario aprenda a estructurar sus peticiones para cualquier IA.
*   **Meta Final**: Lograr una aplicación MVP estable y funcional, generada mediante este flujo.

## Restricciones y Exclusiones (Scope Out)
*   **Exclusión de Responsabilidad**: El sistema **NO** está diseñado ni permitido para dominios críticos donde el error de IA pueda ser fatal o ilegal:
    *   Medicina / Salud.
    *   Legal / Jurídico.
    *   Juegos de Azar / Casinos.
*   **Plataforma**: Windows Desktop (.exe).
