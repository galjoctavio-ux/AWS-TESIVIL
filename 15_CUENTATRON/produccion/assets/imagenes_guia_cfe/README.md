# Imágenes de Ayuda para Recibos CFE

Esta carpeta contiene imágenes de ayuda visual para el flujo de onboarding.

## Imágenes Requeridas

| Archivo | Propósito | Estado |
|---------|-----------|--------|
| `recibo-tarifa.png` | Dónde ubicar tipo de tarifa en recibo CFE | ⬜ Por crear |
| `recibo-fecha-corte.png` | Dónde ubicar fecha de corte | ⬜ Por crear |
| `recibo-consumo.png` | Dónde ubicar consumo (kWh) | ⬜ Por crear |
| `recibo-lectura-cierre.png` | Dónde ubicar lectura de cierre | ⬜ Por crear |
| `medidor-lectura.png` | Cómo leer medidor físico | ⬜ Por crear |

## Especificaciones

- **Formato:** PNG o WebP
- **Resolución:** Mínimo 720x480 px
- **Idioma:** Español MX
- **Estilo:** Fotografías con anotaciones claras

## Uso

Estas imágenes se muestran en modales de ayuda durante el onboarding cuando el usuario toca el ícono "?" junto a cada campo.

## Referencia

- UXUI-016, UXUI-023 a UXUI-027
