# Imágenes de Parámetros - Engine Module

Coloca aquí las imágenes de parámetros (lentes, iluminación, técnicas).

## Archivos requeridos

### Lentes (Ruta A - Fotográfico)
| Archivo | Parámetro | Dimensiones |
|---------|-----------|-------------|
| `param_lens_wide.webp` | Gran Angular | 160×160 px |
| `param_lens_macro.webp` | Macro | 160×160 px |
| `param_lens_bokeh.webp` | Bokeh | 160×160 px |
| `param_lens_drone.webp` | Vista Aérea | 160×160 px |

### Iluminación (Ambas rutas)
| Archivo | Parámetro | Dimensiones |
|---------|-----------|-------------|
| `param_light_natural.webp` | Natural | 160×160 px |
| `param_light_studio.webp` | Estudio | 160×160 px |
| `param_light_golden.webp` | Golden Hour | 160×160 px |
| `param_light_neon.webp` | Neón | 160×160 px |
| `param_light_dramatic.webp` | Dramático | 160×160 px |

### Técnicas (Ruta B - Artístico)
| Archivo | Parámetro | Dimensiones |
|---------|-----------|-------------|
| `param_tech_fine_lines.webp` | Líneas Finas | 160×160 px |
| `param_tech_vibrant.webp` | Vibrante | 160×160 px |
| `param_tech_glow.webp` | Brillo Neón | 160×160 px |

## Especificaciones
- **Formato**: WebP
- **Peso máximo**: 20KB cada una
- **Forma**: Cuadrado (se mostrará en chips)
