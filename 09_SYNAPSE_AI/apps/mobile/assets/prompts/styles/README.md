# Imágenes de Estilos - Engine Module

Coloca aquí las 8 imágenes de estilos para el selector del Prompt Generator.

## Archivos requeridos

| Archivo | Estilo | Dimensiones |
|---------|--------|-------------|
| `style_fotorealismo.webp` | Fotorealismo | 400×300 px |
| `style_arquitectura.webp` | Arquitectura | 400×300 px |
| `style_minimalismo.webp` | Minimalismo | 400×300 px |
| `style_anime.webp` | Anime | 400×300 px |
| `style_3d_pixar.webp` | 3D Pixar | 400×300 px |
| `style_cyberpunk.webp` | Cyberpunk | 400×300 px |
| `style_oleo.webp` | Óleo Clásico | 400×300 px |
| `style_arte_digital.webp` | Arte Digital | 400×300 px |

## Especificaciones
- **Formato**: WebP (mejor compresión)
- **Peso máximo**: 50KB cada una
- **Aspect ratio**: 4:3
- **Estilo visual**: Mismo filtro/tono para verse premium
